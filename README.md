**Features**
* MQTT Broker Install
* Custom Port Dashboard Monitoring
* Bandwidth Monitoring
* CPU and RAM Monitoring

**Support OS**
* Ubuntu 18.04
* Ubuntu 20.04
* Ubuntu 22.04
* Ubuntu 24.04

**Clone Repository**
```
git clone https://github.com/anggrdwjy/mqtt-tools-v2
cd mqtt-tools-v2
chmod -R 777 *
```

**Running Program**
```
./mqtt-tools-v2.sh
```
