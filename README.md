## This Tools Automation Setup For Build MQTT Server

**Features**
* MQTT Broker Install
* Custom Port Dashboard Monitoring
* Bandwidth Monitoring
* CPU and RAM Monitoring

**Clone Repository**
```
git clone https://github.com/anggrdwjy/mqtt-tools-v2
cd mqtt-tools-v2
chmod -R 777 *
```

**Running Program**
```
./mqtt-tools-v2
```
